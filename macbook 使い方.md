# macbook 使い方

⌘ = command
⌥ = option
⇧ = shift
⌃ = control

## キー

| 一般          | macbook              |
| :------------ | :------------------- |
| Home キー     | command ＋ ←         |
| End キー      | command ＋ →         |
| PageUp キー   | command ＋ ↑         |
| PageDown キー | command ＋ ↓         |
| delete        | fn + delete or ⌃ + D |

## ショートカット

| キー          | 効果                                  |
| :------------ | :------------------------------------ |
| ⌘ + M         | ウィンドウを Dock に格納              |
| ⌘ + Tab       | アプリ切り替え                        |
| ⌘ + ⇧ + 3     | スクリーンショット（全画面）          |
| ⌘ + ⇧ + 4     | スクリーンショット（範囲指定）        |
| ⌘ + ^ + F     | フルスクリーン オン/オフ              |
| ⌘ + Z         | もどる                                |
| ⌘ + ⇧ + Z     | 進む                                  |
| ⌘ + ⇧ + .     | finder で隠しファイルとフォルダを表示 |
| ⌘ + ⌥ + space | finder を起動                         |
