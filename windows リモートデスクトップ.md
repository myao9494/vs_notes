# windows リモートデスクトップ

## mac の設定

「Microsoft Remote Desktop」をインストール

## windows 側

1. 「コントロールパネル」を起動
1. 「システムとセキュリティ」をクリック
1. 「リモートアクセスの許可」をクリック
1. 「システムのプロパティ」の「リモート」タブ表示
1. 「このコンピューターへのリモート接続を許可する」を選択
1. cmd で IP アドレスを調べておく `ipconfig all`
1. 設定 → USER でアクセスするユーザ名を調べておく `****@outlook.jp`

## アプリから接続

1. add PC
1. IP アドレスを入力
1. アクセスすると、ユーザ名とパスワードを聞かれるので、入力する
